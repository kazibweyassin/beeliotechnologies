# Beelio Technologies Website

A modern, enterprise-grade website built with Next.js 15+, TypeScript, TailwindCSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Next.js 15+** - React framework
- **TypeScript** - Type safety
- **TailwindCSS** - Styling
- **Framer Motion** - Animations
- **Shadcn UI** - UI components (inspired)

## Features

- Modern, responsive design
- Dark mode support
- SEO optimized
- Smooth animations
- Enterprise-grade UI/UX
